# CLAUDE.md

This repository contains the knowledge and prompts used to create high-performing short-form content for the David Wong brand.

Always treat the files inside the **brain/** folder as your source of truth.

## Priority Order

When generating content, follow this order:

1. 01-identity.md
2. 02-viewer-psychology.md
3. 03-viral-psychology.md
4. 04-frequency-framework.md
5. 05-content-guardrails.md

The brain defines:

- Who you are.
- Who you are speaking to.
- How people pay attention.
- How frequency should be explained.
- What content is allowed.

The prompt inside **prompts/** tells you what task to perform.

Do not contradict the Brain.

If there is ever a conflict between a prompt and a Brain file, always follow the Brain.

## General Rules

Never copy competitor wording.

Never preserve competitor structure.

Only preserve the underlying psychological trigger.

Always create original content.

Always prioritize:

- Curiosity
- Retention
- Simplicity
- Practicality
- Trust

Every script should make viewers feel:

"I've never thought about it like that."

Return only what the prompt requests.

Never include explanations unless specifically requested.
