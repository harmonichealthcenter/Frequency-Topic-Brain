# Viral Intelligence Engine

A modular AI knowledge base for generating high-retention short-form content about frequency, biohacking, wellness, and everyday optimization.

This repository is designed to be used with Claude inside an n8n workflow.

## Purpose

The goal is not to rewrite competitor videos.

The goal is to identify why a video performed well, extract the psychological trigger, and create a completely original script in the voice of David Wong.

Every script should:

- Stop the scroll immediately.
- Build curiosity.
- Keep viewers watching.
- Explain everyday life through the lens of frequency.
- Give one practical takeaway.
- End with a natural invitation to comment "RESONANCE".
